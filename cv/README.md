# CV Files for Evaluation

## Files in this folder

| File | Description | Status |
|------|-------------|--------|
| `cv_primary.txt` | Main CV — used in all analyses | Add manually |
| `cv_paraphrased.txt` | Same content, reworded — robustness test | Add manually |
| `cv_synthetic_control.txt` | Fictional different-domain CV (e.g. UX designer) | To be generated |

## How to add your CV

1. Export your CV as plain text (copy-paste from Word/PDF into a .txt file)
2. Save as `cv_primary.txt` in this folder
3. For paraphrased version: reword ~30% of sentences, keep all facts the same

## Synthetic control CV

Will be generated by Claude — a fictional UX Designer profile.
Used to verify the system correctly gives low scores for irrelevant profiles.
